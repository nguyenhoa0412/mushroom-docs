##########
 required
##########

.. list-table::
   :header-rows: 0
   :stub-columns: 1

   -  -  Type
      -  boolean
   -  -  Required
      -  No
   -  -  Default
      -  false

Flag that determines that the value is required or not. **required**
means existed (has value) and not null.

**Example:**

.. code:: javascript

   {
     "required": true,
     ...
   }
