##########
 codeHint
##########

.. list-table::
   :header-rows: 0
   :stub-columns: 1

   -  -  Type
      -  object
   -  -  Required
      -  No

Specify the code hint for type of this field.

**Example:**

.. code:: javascript

   {
     "codeHint": { // optional
       "type": { // optional
         "csharp": "Abc" // optional
       }
     },
     ...
   }
