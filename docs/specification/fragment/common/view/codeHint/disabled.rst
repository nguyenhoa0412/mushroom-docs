##########
 disabled
##########

.. list-table::
   :header-rows: 0
   :stub-columns: 1

   -  -  Type
      -  boolean
   -  -  Required
      -  No
   -  -  Default
      -  false

Flag that determines that the can be generate code for this view or not.

**Example:**

.. code:: javascript

   {
     "codeHint": {
       "disabled": false, // optional, default: false
       ...
     },
     ...
   }
