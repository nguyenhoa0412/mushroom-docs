#########
 isArray
#########

.. list-table::
   :header-rows: 0
   :stub-columns: 1

   -  -  Type
      -  boolean
   -  -  Required
      -  No
   -  -  Default
      -  false

Flag that determines that the type of this parameter is an array or not.

**Example:**

.. code:: javascript

   {
     "parameters": [
       {
         "isArray": true,
         ...
       },
       ...
     ],
     ...
   }
